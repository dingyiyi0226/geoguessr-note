# Vegetation

You can use the [iNaturalist](https://www.inaturalist.org/observations) website to search for the distribution of different tree species.

For **Europe** and **North America**, the most useful vegetation types are coniferous trees. Check out:
- [Europe](europe/)
- [Mediterranean Pines](mediterranean-pines/)
- [North America](north-america/)

For **Latin America**, the most useful vegetation types are palms, cacti, and agaves. Check out:
- [Palm](palm/)
- [Cactus](cactus/)
- [Agave](agave/)
